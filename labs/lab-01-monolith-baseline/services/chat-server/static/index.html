Lab 01: Stateful Monolith

Collapse sections to inspect protocol, memory, and locking behavior.

Upgrade Handshake
Browser upgrades HTTP to WebSocket with 101 Switching Protocols. Each open socket consumes file descriptors and one read-loop goroutine.
Heap Topography
Active clients are stored in map[*websocket.Conn]bool. State is local and volatile: process death means total state loss.
Lock Contention
Broadcast takes a global sync.Mutex and iterates all clients. One slow receiver can stall delivery for everyone else.

Right pane shows live socket frames plus metric snapshots: current connections, end-to-end broadcast latency, and node id.
